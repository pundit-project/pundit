Detection code (bitfield):
10000000 No problems
01000000 Excessive Delays (Range Tomo)
00100000 Excessive Losses (Boolean Tomo)
00010000 Congestion (Range Tomo)
00001000 Route change (Boolean Tomo)
00000100 Route instability (Boolean Tomo)
00000010 Extensive reordering (Boolean Tomo)
00000001 End-host context switch

val2: for upper bound if Range Tomo
